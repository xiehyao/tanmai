# 探脉小程序

探脉是一个社交匹配小程序，通过AI智能匹配帮助用户建立有价值的人脉关系。

## 项目结构

```
TanMai/
├── frontend/          # uni-app前端
├── backend/           # Python FastAPI后端
└── docs/              # 文档
```

## 核心功能

1. 名片系统：公开名片 + 隐藏信息
2. AI语音互动录入：语音对话方式录入信息
3. 自我简介卡片：多场景非正式简介卡片
4. 周边朋友分布地图：基于地理位置的好友匹配
5. 智能匹配：AI自动匹配推荐
6. AI助手：付费智能匹配助手
7. AI人脉维护小助理：智能提醒维护
8. 好友信息更新智能提醒：联动机会提醒
9. 探脉个人商务秘书：企业微信个人助理

## 技术栈

- 前端：uni-app + Vue3
- 后端：Python FastAPI
- 数据库：PostgreSQL + Redis
- AI服务：阿里云通义千问
- 地图：腾讯地图
